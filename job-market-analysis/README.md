# Job Market Skills Analysis

## Overview
This project analyzes data analyst job postings to identify in-demand skills, common skill combinations, job locations, and salary trends.

## Tools Used
- Python (pandas, numpy)
- Data Visualization (matplotlib, seaborn)
- Google Colab

## Key Insights
- SQL is the most in-demand skill across job postings
- Excel remains highly relevant in data analyst roles
- Python is valuable but not always required
- Job opportunities are concentrated in major cities
- Salaries are primarily within the $50K–$80K range
- Employers expect combinations of technical skills rather than a single tool
